# About Termmap

The PDS4 Terminological Mapping tool (Termmap) maps a PDS3 keyword to its equivalent or closely matched counterpart in PDS4. This application also provides a web service to search for pds4 records using a pds3 keyword or vice versa.

Please send comments, change requests and bug reports to the [PDS Operator](mailto:pds_operator@jpl.nasa.gov) at pds_operator@jpl.nasa.gov.
